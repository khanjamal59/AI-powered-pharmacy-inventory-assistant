**AI-Powered Pharmacy Inventory Chatbot**

An intelligent AI chatbot that retrieves real-time pharmacy inventory information from an Excel-based inventory system using Natural Language Processing (NLP) and Large Language Models (LLMs).

Instead of manually searching spreadsheets, pharmacists can simply ask questions like:
"How many Paracetamol tablets are in stock?"

and receive an accurate response based on the latest inventory data.


**Basic architecture**
    
    User Question

          ↓

    Understand Question

          ↓

      Search Inventory

          ↓

      Generate Answer
